# Titanic Survival Analysis (Exploratory Data Analysis)

## Overview

The Titanic Survival Analysis project explores the Titanic passenger dataset to identify the factors that influenced passenger survival. Using Python and Exploratory Data Analysis (EDA) techniques, the project uncovers patterns, trends, and relationships within the data through statistical analysis and data visualization.

The project demonstrates practical data analysis skills, including data cleaning, feature exploration, and visualization to generate meaningful business insights from real-world data.

---

## Objectives

- Explore the Titanic dataset
- Clean and preprocess the data
- Identify missing values and handle them appropriately
- Analyze passenger demographics and travel information
- Investigate factors influencing survival
- Create visualizations to communicate findings

---

## Dataset

The dataset contains passenger information, including:

- Passenger ID
- Survival status
- Passenger class (Pclass)
- Name
- Sex
- Age
- Number of siblings/spouses aboard (SibSp)
- Number of parents/children aboard (Parch)
- Ticket number
- Fare
- Cabin
- Embarked port

## Key Insights

- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers were more likely to survive than those in lower classes.
- Younger passengers generally had higher survival rates.
- Passenger class and fare showed a positive relationship with survival.
- Missing values were identified and appropriately handled during preprocessing.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ghamandiashwini/Titanic_Survival_Analysis_EDA-.git
```

2. Navigate to the project folder

```bash
cd Titanic_Survival_Analysis_EDA
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
